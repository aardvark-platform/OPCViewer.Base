## Quickstart

Use argument with base folder

Ctrl + LMB to select a point on surface

## Build Instructions

Clone the repository:
```
git clone https://github.com/aardvark-platform/OpcViewer.git
cd OPCViewer
```

Build the project:
```
build
```
